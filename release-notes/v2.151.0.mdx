## Highlights

- Generating an RMA contract no longer requires a signed PC contract on the associated opportunity. You only need an owner's rep assigned.

## Changes shipped to production

### revive-admin

- **Simplified RMA contract generation** — Previously, you needed both an assigned owner's rep and a signed Professional Contractor (PC) contract on the opportunity before you could generate an RMA contract. The signed PC requirement has been removed. You can now generate an RMA contract as soon as an owner's rep is assigned. If the owner's rep is missing, the **Generate RMA** button remains disabled with a tooltip explaining what is needed. ([#1746](https://github.com/Revive-Home/revive-admin/pull/1746))
