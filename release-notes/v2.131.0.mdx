## Highlights

- Sell 360 renovation plan forms now include a **1st check amount** field with a recommended-max hint based on the as-is price.
- The previous **Agent commission** field has been renamed to **1st transaction commission**.

## Sell 360 estimate input updates

### New field: 1st check amount

The Sell 360 renovation plan form now includes a **1st check amount** currency input. This field captures the initial payment amount for the Sell 360 transaction.

When you enter an as-is price, a **Recommended max** hint appears below the field showing 80% of the as-is value. For example, if the as-is price is $500,000, the hint displays "Recommended max: $400,000." This helps you stay within typical first-check thresholds.

The field appears after the 1st transaction commission field and is only visible for Sell 360 opportunities. The value is saved to the deal and included in the estimate share modal and preview.

### Renamed field: 1st transaction commission

The field previously labeled **Agent commission** is now **1st transaction commission**. The behavior is unchanged — it still captures the expected commission amount for the first transaction. The Sell 360 renovation plan form now shows the following fields in order:

| Field | Description |
|---|---|
| **As-is price** | Current property value before renovation |
| **Renovation budget** | Planned renovation spend |
| **After renovation value** | Projected property value after renovation |
| **Mortgage / liens** | Outstanding debt on the property |
| **1st transaction commission** | Expected commission amount (previously "Agent commission") |
| **1st check amount** | Initial check amount, with a recommended max of 80% of the as-is price |
