## Highlights

- The RMA list page now features tabbed views, inline filters, a summary header, and CSV export for faster triage.
- RMA statuses have been renamed to match production terminology — "Released" is now "Paid", "Rejected" is now "Not Approved", and new intermediate statuses are available.
- Sell 360 estimates support mortgage/liens and agent commission inputs, and you can generate shareable pre-estimate links.

## RMA list redesign

The RMA list page has been reorganized into a tabbed layout so you can focus on the RMAs that need attention.

### Tabs

| Tab | Shows |
|---|---|
| **Pending** | RMAs in Requested, Received, Preparing, or Pending Signatures status |
| **On Hold** | RMAs currently on hold |
| **Ready for Check Run** | Executed RMAs created before the Thursday cutoff |
| **Next Check Run** | Executed RMAs created after the Thursday cutoff |
| **Completed** | Paid or Not Approved RMAs |
| **All** | Every RMA regardless of status |

Each tab displays a badge with the count of matching RMAs. Counts reflect any active column filters.

### Summary header

Three summary cards appear above the tabs:

- **Executed cutoff** — the Friday date used to split executed RMAs between the current and next check run.
- **Check run** — the date of the next scheduled check run (Thursday).
- **Check run total** — the sum of approved (or requested) amounts for RMAs in the "Ready for Check Run" tab.

### CSV export

Click **Export CSV** in the toolbar to download the currently visible RMA data. The export includes name, status, deal, contractor, requestor, dates, amounts, owner's rep, and final-payment flag.

### Default filter

When you open the RMA list, it automatically filters to show RMAs assigned to your owner's rep account.

## Updated RMA statuses

RMA statuses have been renamed to better reflect the actual workflow:

| Old status | New status |
|---|---|
| Drafting | Preparing |
| Sent | Pending Signatures |
| Review | Executed |
| Rejected | Not Approved |
| Released | Paid |

Two new status transitions are available from the detail page and the inline status popover:

- **Received** — marks a Requested RMA as received, confirming you have the physical or digital documents.
- **Resume** — moves an On Hold RMA back to Executed status.

Hold and Paid actions are disabled until an RMA reaches the Executed stage.

## Sell 360 estimate inputs

When working with a Sell 360 opportunity, the renovation plan form now includes two additional fields:

- **Mortgage / liens** — the outstanding debt on the property.
- **Agent commission** — the expected agent commission amount.

These values feed into the Sell 360 transaction summary and shareable pre-estimate links, giving homeowners a clearer picture of projected proceeds.

## Fixes

- RMA page defaults to the Pending tab and resets filters correctly when switching tabs.
- Column filters (owner's rep, contractor, opportunity, status) now use stable identifiers for consistent results.

## Breaking changes

No breaking changes.
