## What this page is

This is the landing page for the auto-generated API reference. It’s written for internal engineers working across `revive-api`, `revive-admin`, `revive-dashboard`, and `revive-mobile`.

Use this page to:

- pick the right base URL for your environment
- confirm authentication requirements
- copy/paste a first request
- understand basic conventions (errors, pagination)

## Base URLs

The API reference is generated from `openapi.json`, which defines these servers:

- Production: `https://revive-api-v2-production.herokuapp.com`
- Staging: `https://revive-api-v2-staging.herokuapp.com`
- Local: `http://localhost:3001`

Paths are typically versioned under `/v1` and `/v2`.

## Authentication

Most endpoints require a Firebase JWT passed via the `Authorization` header.

```bash
Authorization: Bearer <firebase_jwt>
```

In the OpenAPI spec this is represented as `BearerAuth` with `bearerFormat: Firebase JWT`.

## Make your first request

Start with the unauthenticated health check:

<CodeGroup>
```bash cURL
curl -sS https://revive-api-v2-production.herokuapp.com/health
```

```javascript JavaScript
const res = await fetch('https://revive-api-v2-production.herokuapp.com/health');
const data = await res.json();
console.log(data);
// { status: 'ok', version: '2.152.0', uptime: 86400 }
```

```python Python
import requests

resp = requests.get('https://revive-api-v2-production.herokuapp.com/health')
print(resp.json())
# {'status': 'ok', 'version': '2.152.0', 'uptime': 86400}
```
</CodeGroup>

Then try an authenticated endpoint using your Firebase JWT:

<CodeGroup>
```bash cURL
curl -sS -H "Authorization: Bearer $TOKEN" \
  https://revive-api-v2-production.herokuapp.com/v1/contacts/?limit=10
```

```javascript JavaScript
const res = await fetch(
  'https://revive-api-v2-production.herokuapp.com/v1/contacts/?limit=10',
  { headers: { Authorization: `Bearer ${token}` } }
);
const { data, total } = await res.json();
```

```python Python
import requests

resp = requests.get(
    'https://revive-api-v2-production.herokuapp.com/v1/contacts/',
    params={'limit': 10},
    headers={'Authorization': f'Bearer {token}'},
)
data = resp.json()
```
</CodeGroup>
